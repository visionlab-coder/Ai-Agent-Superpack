# /pull - Git Pull (Quick)

---

## 1단계: 현재 상태 확인

```bash
git branch --show-current
git status --short
```

커밋되지 않은 변경사항이 있으면 경고:
```
작업 중인 변경사항이 있습니다. stash 후 진행할까요? (Y/n)
```
- Y: `git stash` 후 진행, pull 후 `git stash pop`
- n: 그대로 pull 시도

---

## 2단계: Pull 실행

```bash
git pull origin main
```

---

## 3단계: 출력

### 성공 시

```
════════════════════════════════════════════════════════════════
  Git Pull
════════════════════════════════════════════════════════════════

  브랜치: {branch}
  상태: {up-to-date | N commits pulled}

════════════════════════════════════════════════════════════════
```

### 충돌 시

```
════════════════════════════════════════════════════════════════
  Git Pull
════════════════════════════════════════════════════════════════

  충돌 발생! 수동 해결 필요:
  {충돌 파일 목록}

════════════════════════════════════════════════════════════════
```
